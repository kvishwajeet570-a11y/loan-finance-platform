# package-lock.json

`package-lock.json` is automatically generated by npm.

You do not need to manually create or edit it.

This file is automatically created when running:

```bash
npm install
```

# Purpose of package-lock.json

* Locks dependency versions
* Ensures same dependency installation on every machine
* Prevents dependency conflicts
* Improves production build stability
* Required for deployment environments

---

# Important

Do not delete:

```text
package-lock.json
```

because:

* npm depends on it for dependency consistency
* Deployment platforms use it during installation
* Production builds become more stable and reproducible
